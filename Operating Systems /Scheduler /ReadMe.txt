Test a round robin vs a first come first serve algorithm which may sometimes be used in an operating system. It uses 4 different data sets. 


To run the scheduler program go to 
scheduler > src 

If in the current directory these files exist:

scheduler.java testdata1.txt testdata2.txt testdata3.txt testdata4.txt 

Then run the program with these commands...

$ javac scheduler.java 
$ java scheduler
